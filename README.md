# 🤖 AI Debate Arena

An interactive AI-powered web application where users engage in debates with AI to improve their critical thinking, communication, and confidence.

## 🚀 Features

* 🧠 Chat-based AI debate system
* ⚔️ AI generates counterarguments
* 📊 Performance scoring (Logic, Clarity, Confidence)
* 🎮 Gamification (XP, levels, badges)
* 🌙 Dark mode with animated UI
* 📈 Dashboard to track progress

---

## 🛠️ Tech Stack

* Frontend: React + Tailwind CSS
* Backend: Node.js / Flask
* AI Integration: OpenAI API / Gemini
* Deployment: Netlify (Frontend)

---

## ⚙️ Installation

```bash
git clone https://github.com/Vaishnavi-2126/ai-debate-arena.git
cd ai-debate-arena
npm install
npm start
